# Body Fat Predictor API

This project provides a machine learning-based API to predict body fat percentage based on various body measurements. It uses a Linear Regression model trained on a dataset of body measurements (`bodyfat.csv`) and is served using FastAPI. A frontend is also included to interact with the API.

## Project Structure

- `app.py`: The main FastAPI application. Handles API routes, serves the static frontend, and trains the machine learning model on startup.
- `proj.py`: A data exploration and model evaluation script. Includes exploratory data analysis (EDA), visualizations, and detailed model evaluation metrics.
- `bodyfat.csv`: The dataset used to train the model.
- `static/`: Contains the frontend assets (`index.html`, `main.js`, `style.css`).
- `requirements.txt`: Python dependencies.

## Features

- **Predictive Model**: Uses scikit-learn's `LinearRegression` to predict body fat percentage.
- **RESTful API**: Built with FastAPI, providing endpoints for model information and predictions.
- **Data Visualizations**: `proj.py` generates comprehensive visualizations including cross-validation scores, feature correlations, actual vs. predicted values, residual analysis, and learning curves.
- **Interactive Frontend**: A web interface to input body measurements and receive instant body fat predictions and categorizations.

## Installation and Setup

1. **Clone the repository:**
   ```bash
   git clone <repository_url>
   cd "ML_proj"
   ```

2. **Create a virtual environment (recommended):**
   ```bash
   python -m venv venv
   # On Windows, activate with:
   venv\Scripts\activate
   # On Linux/macOS, activate with:
   source venv/bin/activate
   ```

3. **Install the dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

## Running the Application

To start the FastAPI server, run:

```bash
uvicorn app:app --reload
```

The server will start at `http://127.0.0.1:8000`. 
- You can access the web application by navigating to the root URL in your browser.
- The interactive API documentation (Swagger UI) is automatically generated by FastAPI and available at `http://127.0.0.1:8000/docs`.

## API Endpoints

- `GET /`: Serves the frontend application.
- `GET /api/info`: Returns information about the trained model, including R² score, RMSE, mean cross-validation accuracy, dataset size, and the list of features used.
- `POST /api/predict`: Accepts a JSON payload with body measurements and returns the predicted body fat percentage, category (e.g., "Fit", "Average"), and a brief description.

## Exploratory Data Analysis

Run the `proj.py` script to generate data visualizations and evaluate the model's performance in detail:

```bash
python proj.py
```

This will display plots such as correlation heatmaps and learning curves using `matplotlib` and `seaborn`, and print training/testing performance metrics to the console.
